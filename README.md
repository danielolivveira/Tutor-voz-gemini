# 🎙️ Tutor de Estudos por Voz com Whisper e Gemini

Um assistente de estudos que escuta sua voz, transcreve e responde como um professor didático em português.

## 🚀 Como funciona

1. 🎤 Grava sua pergunta por áudio
2. 🧠 Whisper transcreve o áudio para texto
3. 💬 Gemini responde como um professor
4. 🔊 gTTS converte a resposta em voz

## 🛠️ Tecnologias

- [Whisper](https://github.com/openai/whisper) — transcrição de voz
- [Gemini API](https://ai.google.dev) — resposta inteligente
- [gTTS](https://gtts.readthedocs.io) — síntese de voz
- Google Colab — ambiente de execução

## ▶️ Como usar

1. Abra o notebook no Google Colab
2. Adicione sua `GEMINI_API_KEY` nos Secrets do Colab
3. Execute as células em ordem

## ⚠️ Observação

Não compartilhe sua API Key. Use sempre os Secrets do Colab.
